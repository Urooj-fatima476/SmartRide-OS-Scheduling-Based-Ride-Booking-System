🚖 Built **SmartRide**, a Flask-based Ride Booking & Dispatch System inspired by **Operating System Scheduling Algorithms**.

The project simulates how ride requests can be managed efficiently using classic CPU scheduling concepts:

• Priority Scheduling → Emergency rides handled first
• FCFS (First Come First Served) → Requests processed in arrival order
• Round Robin → Fair ride allocation using time quantum
• SJF (Shortest Job First) → Nearest/shortest rides prioritized

🔧 Tech Stack:
Python, Flask, SQLite, HTML/CSS/JS

✨ Features:

* Real-time ride queue management
* Smart driver allocation
* ETA & fare calculation
* Admin dashboard & analytics
* Driver status tracking
* REST APIs for rides and statistics
* Queue logging system

This project helped me understand how Operating System concepts can be applied in real-world applications like ride-hailing systems.
